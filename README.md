# LeetCode Solutions

This repository contains my **LeetCode solutions** written in Python.  
The goal is to improve problem-solving skills and share clear examples for others.

- Solutions are implemented in Python.  
- Problems are organized by difficulty: **easy / medium / hard**.  
- Each solution includes the problem name, a short explanation, and example usage.  
- Original LeetCode links are included in each file.

---

## Folders

- `easy/` → Easy problems  
- `medium/` → Medium problems  
- `hard/` → Hard problems
